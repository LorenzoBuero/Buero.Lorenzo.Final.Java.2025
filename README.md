# Diseño Inteligente
### Aplicación de diseño de cartas dedicada a un juego de cartas de mi creación llamado "Evolución"

El objetivo de esta aplicación es el de proporcionar un ambiente que simplifique el diseño de cartas de Evolución, brindándome herramientas que me faciliten:
- La Alta, Baja y Modificación de los datos correspondientes a las cartas de forma eficiente y automatizada.
- Una visualización de las cartas cómoda y localizada.
- El buen diseño y balanceo de las cartas a nivel mecánico gracias a multiples herramientas provistas por esta aplicación.
